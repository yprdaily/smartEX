# スマートEX 予約連携システム (SmartEX Reservation Sync System)

このリポジトリは、Gmailで受信した[スマートEX](https://smart-ex.jp/)の新幹線予約関連のメールを自動で解析し、Googleスプレッドシートの管理台帳への記帳と、Googleカレンダーへの予定登録を自動化するGoogle Apps Script (GAS) のソースコードです。

## 主な機能

* **メール自動解析**: `yoyaku@expy.jp` から届く予約完了・予約変更・払戻・ICカード指定メールを一定間隔で読み込み、本文をパースして乗車日、区間、金額、対象者などを自動抽出します。
* **Googleスプレッドシート連携**: 抽出したデータを経費精算用データや集計用メインシート、システム稼働を管理する管理シート（`スマートEX管理`シートなど）に自動的に書き込みます。
* **Googleカレンダー連携**: 予約された乗車日時をもとに、乗車対象者のGoogleカレンダーへ自動的に予定を作成します。予定の変更や払戻し（キャンセル）が発生した場合は、カレンダーの予定も自動的に更新・削除されます。
* **自動分類と例外管理**:
  * 処理が完了したメールには「処理済」ラベル、パースエラーや要確認のメールには「要確認」等の専用ラベルを付与し、Gmail上で状態を視覚的に管理できます。
  * ICカード番号に基づく従業員の特定機能や、未登録カードの検知（総務への通知）機能を備えています。
* **安全装置（ブレーカー機能）**: 連続してエラーやパース異常を検知した場合は自動で処理を一時停止し、無限ループや誤作動を防ぐ保護機構が組み込まれています。

## 導入方法・セットアップ

本スクリプトを組織・チーム等でご利用いただくには、以下の初期設定が必要です。

1. **GASプロジェクトへの配置**:
   * Google Apps Scriptのプロジェクトを新規作成し、本リポジトリの `スマートEX.txt` の内容をコピーして `.gs` ファイル（例: `main.gs`）として保存してください。
2. **環境変数の設定 (CONFIGの編集)**:
   * スクリプト上部にある `CONFIG` オブジェクト内の設定値を、皆様の環境に合わせて書き換えてください。
     * `MAIN_SPREADSHEET_ID`: メインデータが集約されるスプレッドシートのID
     * `LOG_SPREADSHEET_ID`: 管理・ログ用途のスプレッドシートID
     * `SOUMU_MAIL` / `ADMIN_MAIL`: システム通知先のメールアドレス
     * `CALENDAR_ID`: カレンダー予定の同期先IDなど
   * *※リポジトリ上のソースコードでは、セキュリティの観点からダミー文字列 (`YOUR_MAIN_SPREADSHEET_ID_HERE`等) にマスキングされています。*
3. **シートの準備**:
   * スクリプト内で指定されている数々のシート（`社員情報`、`スマートEX管理`など）を上記のスプレッドシート内に作成し、指定のヘッダーを用意してください。
4. **Google Workspace (API) 認証の許可**:
   * スクリプトの初回実行時に、Gmail, Spreadsheet, Calendar各APIへのアクセス権限を許可する必要があります。
5. **定期実行トリガーの設定**:
   * GASの「トリガー」より、本スクリプトの起点（例: 時間主導型のスケジュール機能など）を定期的に動作させるように設定します。
   * また、スプレッドシートのメニューバーに専用UIを展開するための `onOpen` 関数も組み込まれていますので、必要に応じて活用してください。
